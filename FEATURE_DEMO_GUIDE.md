# Edit Program Feature - User Guide

## How to Use the New Edit Program Feature

### Step 1: Navigate to Your Programs
1. Open the Fitness Tracker app
2. Tap on **"Start Workout"** from the Home screen
3. Select **"My Programs"**

### Step 2: Open a Program for Editing
1. You'll see a grid of your workout programs
2. **Tap on any program card** to open the edit view
3. The edit screen will open in a modal sheet

### Step 3: Edit Program Name
- The program name appears at the top in an editable text field
- Tap on it to change the name
- The changes are saved when you tap "Save Changes"

### Step 4: Add Exercises to Your Program

#### If No Exercises Yet:
- You'll see an empty state with a message "No exercises yet"
- Tap the **"Add Exercise"** button in the center

#### If Exercises Already Exist:
- Scroll to the bottom of the exercise list
- Tap the **"Add Exercise"** button

#### In the Exercise Picker:
1. Use the search bar to find specific exercises
2. Browse through the available exercises
3. Tap on an exercise to add it to your program
4. The exercise will appear in your program with default settings:
   - 3 sets
   - 10 reps
   - 0 kg weight
   - 60 seconds rest

### Step 5: Customize Exercise Settings

#### Expand an Exercise:
1. Tap on any exercise card to expand it
2. The card will smoothly expand to show all settings

#### Adjust Settings:
Each exercise has the following adjustable parameters:

**Sets (SÃ¤tze)**
- Use the **minus (-)** button to decrease
- Use the **plus (+)** button to increase
- Shows current number in the middle

**Reps (Wiederholungen)**
- Use the **minus (-)** button to decrease
- Use the **plus (+)** button to increase
- Shows current number in the middle

**Weight (Gewicht)**
- Measured in kilograms (kg)
- Use the **minus (-)** button to decrease
- Use the **plus (+)** button to increase
- Start at 0 kg if you're using bodyweight

**Rest (Pause)**
- Measured in seconds (sec)
- Use the **minus (-)** button to decrease
- Use the **plus (+)** button to increase
- Default is 60 seconds

**Notes (Notizen)**
- Optional text field for additional information
- Can include form tips, variations, or reminders
- Multi-line support
- Example: "Focus on controlled movement" or "Use resistance band"

### Step 6: Remove an Exercise
1. Expand the exercise card by tapping on it
2. Scroll to the bottom of the expanded card
3. Tap the red **"Remove Exercise"** button
4. Confirm the removal in the alert dialog
5. The exercise will be removed from the program

### Step 7: Save Your Changes
1. After making all desired changes, tap **"Save Changes"** at the bottom
2. All your modifications will be saved automatically
3. The edit view will close and return to the program list
4. Your changes are now persistent and will be there next time you open the program

### Step 8: Cancel Without Saving
- If you want to discard your changes, tap **"Cancel"** in the top-left corner
- None of your edits will be saved
- The program will remain as it was before you opened the edit view

## Tips and Tricks

### Efficient Workout Planning
- **Set realistic weights**: Start lower and increase gradually
- **Rest periods matter**: 
  - 30-60 seconds for endurance
  - 60-90 seconds for hypertrophy
  - 2-5 minutes for strength
- **Use notes wisely**: Add specific cues or technique reminders

### Exercise Organization
- Add exercises in the order you plan to perform them
- Group similar exercises together (e.g., all chest exercises)
- Consider adding warm-up exercises first

### Weight Tracking
- Start with 0 kg for bodyweight exercises
- Record the actual weight you plan to use
- Update as you progress over time

### Using Notes Effectively
Examples of useful notes:
- "Tempo: 3-1-1-1"
- "Use decline bench"
- "Superset with pull-ups"
- "Focus on mind-muscle connection"
- "Partial reps on last set"

## Troubleshooting

**Q: My changes aren't saving**
- Make sure you tap "Save Changes" before closing the view
- If you tap "Cancel", changes are discarded

**Q: I can't find an exercise**
- Use the search bar in the exercise picker
- Check if it's already added to your program
- Exercises already in the program won't appear in the picker

**Q: The minus button is grayed out**
- You've reached the minimum value (usually 0)
- You can't have negative sets, reps, or rest time

**Q: How do I reorder exercises?**
- Currently, you'll need to remove and re-add exercises in the desired order
- Drag-and-drop reordering may be added in a future update
